# P-1001 공기 필터 교체 절차 (Air Filter Replacement)

## 기본 정보
- 절차코드: P-1001
- 명칭: 공조기(AHU/FCU) 공기 필터 교체
- 문서 종류: 점검 및 수리 절차서
- 적용 기기: AHU, FCU

## 교체 주기
- 프리필터: 1~3개월 (외기 분진 농도에 따라)
- 미디엄/헤파필터: 6~12개월 또는 차압 경보 시
- 차압계가 설정값 초과 시 주기와 무관하게 교체

## 교체 절차
1. 해당 공조기 송풍기 정지 및 전원 차단(LOTO)
2. 필터 접근 도어 개방, 기존 필터 탈거
3. 프레임/가스켓 청소, 누기 여부 확인
4. 신규 필터를 공기 흐름 방향(화살표)에 맞춰 장착
5. 도어 닫고 차압계 0점 확인 후 송풍기 재기동
6. 교체 일자/필터 등급 대장 기록

## 비고
필터 막힘은 풍량 저하와 급기온도 이상으로 이어질 수 있다. 차압 추세를 모니터링하면 교체 시점을 예측할 수 있다.
