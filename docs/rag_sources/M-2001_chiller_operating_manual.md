# M-2001 냉동기 운전 매뉴얼 (Chiller Operating Manual)

## 기본 정보
- 매뉴얼코드: M-2001
- 명칭: 수냉식 냉동기(Chiller) 정상 운전 매뉴얼
- 문서 종류: 기기 운전 매뉴얼
- 적용 기기: 수냉식 냉동기

## 정상 운전 범위
- 냉수 출구 온도: 7°C (설정), 허용 ±1.5°C
- 냉수 입출구 차온: 5°C 내외
- 응축기 입구 냉각수 온도: 32°C 이하 권장
- 증발압력/응축압력: 기종 사양서 기준 범위 내

## 기동 및 정지
1. 냉각수 펌프 → 냉수 펌프 → 냉각탑 순으로 보조기기 선기동
2. 유온/유압 정상 확인 후 압축기 기동
3. 정지 시 역순, 압축기 정지 후 보조기기 일정 시간 후 정지

## 정기 점검 항목
- 일간: 운전 압력/온도, 전류, 이상 소음
- 주간: 냉각수 수질, 스트레이너 차압
- 연간: 냉매량, 윤활유, 전동기 절연

## 비고
출구 냉수 온도가 설정값 대비 지속 상승하거나 압력이 비정상이면 부하/냉매/열교환 상태를 함께 점검한다.
