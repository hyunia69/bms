# P-3001 순환펌프 유량 저하 / 캐비테이션 점검 절차 (Pump Flow Loss / Cavitation)

## 기본 정보
- 절차코드: P-3001
- 명칭: 순환펌프 유량 저하 및 캐비테이션 점검
- 문서 종류: 점검 및 수리 절차서
- 적용 기기: 순환펌프(Pump)

## 증상
- 유량이 정상 대비 지속 하락하고 전류가 변동(불안정)하며 동반된다
- 흡입측 압력 저하, 소음·진동 증가 (캐비테이션 신호)
- 토출압력 불안정, 펌프 과열

## 가능한 원인
- 흡입측 스트레이너 막힘 또는 배관 폐색
- 흡입 양정 부족·공기 혼입 (캐비테이션)
- 임펠러 마모·이물, 커플링/베어링 이상
- 밸브 개도 오류 또는 계통 차압 변화

## 점검 절차
1. 흡입측 스트레이너 차압·이물 점검 및 청소
2. 흡입 배관 공기 혼입·누기 확인 후 에어 벤트
3. 토출/흡입 압력 게이지로 양정 확인, 펌프 성능곡선과 비교
4. 임펠러·커플링·베어링 상태 점검 (소음·진동)
5. 전류 변동폭 확인 — 일정 방향이 아닌 변동이면 캐비테이션 의심
6. 밸브 개도·계통 차압 재설정

## 비고
유량 저하는 단일 수치보다 추세로 판단한다. 전류가 한 방향으로 가지 않고 변동(불안정)하는 것이 캐비테이션의 특징이다. 냉각수 순환펌프이면 냉각탑·냉동기 동반 신호를 함께 본다 (냉각수 계통 복합 고장 → P-4001 참조).
