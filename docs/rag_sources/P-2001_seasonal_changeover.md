# P-2001 계절 운전 전환 절차 (Seasonal Operation Changeover)

## 기본 정보
- 절차코드: P-2001
- 명칭: 냉방 / 중간기 / 난방 운전 모드 전환
- 문서 종류: 점검 및 수리 절차서
- 적용 기기: AHU, 냉동기(Chiller), 시스템 전반

## 적용 시기
- 외기온도 추세가 전환 기준을 넘을 때 (예: 중간기 → 냉방, 난방 → 중간기)
- 환절기 새벽·주간 외기 변동이 큰 시기에는 운전 스케줄을 재점검
- 냉난방 동시 요구가 잦아지면 중간기(외기냉방) 모드를 검토

## 전환 절차
1. 현재 운전 모드와 시간대별 설정 스케줄 확인
2. 냉방 전환: 냉각수 펌프 → 냉수 펌프 → 냉각탑 → 압축기 순으로 기동 (M-2001), 냉수 설정온도 적용
3. 난방 전환: 온수/열원 계통으로 절체하고 냉방 계통 정지
4. 중간기(외기냉방): 외기댐퍼 개도 로직 활성화, 기계식 냉방 최소화
5. 전환 후 급기/환기온도가 설정값에 수렴하는지 1~2주기 모니터링

## 점검 항목
- 모드별 설정값(급기온도, 외기댐퍼 최소개도)이 계절에 맞는지
- 스케줄러가 새벽·주말에 잘못된 모드로 고정돼 있지 않은지
- 전환 직후 급기온도 상한 초과(E-1002) 발생 여부

## 비고
새벽 급기온도 급상승의 흔한 근인은 계절 전환 누락이다. 전환은 외기 추세로 판단하되, 단기 변동에 과민 반응하지 않도록 이동평균 등 이력으로 결정한다.
